# Naveen Bharat — Student Guide

Welcome! Ye guide aapko app ke har feature ka use sikhayegi — sign up se lekar live class tak.

---

## 1. Getting started

### Sign up
1. Home par "Sign Up" button dabaiye.
2. Naam, email, mobile aur ek strong password daliye.
3. Email par aayi verification link kholiye — bas, account ready.

### Login
- Pehle se account hai? "Login" → email + password.
- Password bhool gaye? "Forgot Password" → email par reset link milega.

### Install as app
- Phone browser me site khol kar **Install** page par jaiye (menu se).
- Android: "Add to Home Screen" prompt par OK dabaiye.
- iPhone (Safari): Share → "Add to Home Screen".
- Ab Naveen Bharat aapke home screen par ek app ki tarah dikhega.

---

## 2. Dashboard

Login ke baad seedha Dashboard khulta hai. Yahaan se aap pahunch sakte ho:
- **My Courses** — kharide hue courses
- **Browse Courses** — naye courses dhundhne ke liye
- **Attendance, Reports, Timetable, Notices**
- **Books library, Notes, Doubts**
- **Live Classes**
- **Profile, Settings**

Bottom nav bar (Home / Courses / My Courses / Downloads / Profile) se sab kuch ek tap door hai.

---

## 3. Courses kaise kharide

1. Bottom nav me **Courses** dabaiye.
2. Course tile par tap karke detail page kholiye — syllabus, faculty, kya kya milega sab dikh jayega.
3. **Buy Course** button dabaiye.
4. Razorpay checkout khulega — UPI / Card / NetBanking / Wallet — koi bhi method choose karo.
5. Payment ke baad app khud course aapke "My Courses" me daal dega. Receipt email par milegi.

> Tip: kabhi-kabhi UPI app se wapas aane me 5-10 second lagte hain. Wait karo, manually refresh mat karna.

---

## 4. Lesson (video) kaise dekhe

1. **My Courses** → course tap → chapter chuno → lesson tap.
2. Custom player khulega — saari controls upar/neeche dikhti hain.

### Player controls
| Control | Kaam |
|---|---|
| Play / Pause | Tap (ya keyboard se space) |
| Seek forward 10s | Right arrow / double-tap right side |
| Seek backward 10s | Left arrow / double-tap left side |
| Volume | Slider / M se mute |
| Speed | Settings ⚙ → 0.5x – 2x |
| **Rotate + Fullscreen** | **Rotate (⟳) icon — ek tap me landscape + fullscreen dono** |
| Exit fullscreen | Same icon dobara (ya hardware back / ESC) — phone wapas portrait ho jayega |

Tap karke controls overlay show/hide karo. Right-click / context menu band hai (security).

### Offline ke liye
Jin lessons par "Downloads" icon dikhta hai unhe **Downloads** tab me save kar sakte ho. Bina internet ke bhi play hoga.

---

## 5. Quizzes, DPP aur Notes

- **All Tests** se quiz attempt karo. Timer chalu hota hai, submit ke turant baad result + answer key.
- **DPP (Daily Practice)** — har chapter ke saath. PDF kholo, hal karo.
- **Notes** — class ke baad teacher ke notes / aapke khud ke notes ek hi jagah.

---

## 6. Doubts (sawal puchhna)

1. Lesson page ke neeche **Doubts** section.
2. Sawal type karo, screenshot bhi add kar sakte ho.
3. Teacher reply karega — notification se pata chalega.

---

## 7. Live Classes

- Schedule **Timetable** me dikhega.
- Class start hone se 5 min pehle "Join" button active ho jayega.
- Camera / mic optional — chat se bhi sawal puchh sakte ho.

---

## 8. Profile aur Reports

- **Profile** se naam, photo, mobile update kar sakte ho.
- **Reports** me apna attendance %, quiz score progress aur chapter-wise mastery dikhta hai.
- **Settings** se notifications, theme (Light/Dark) aur language change karo.

---

## 9. Back button (Android app)

- Kahin bhi back dabane par parent folder par jaate ho (Dashboard nahi).
- Dashboard par back dabaoge to ek hint aayegi — **"Press back again to exit"**.
- 2 second ke andar dobara back dabane par app band ho jayega.

---

## 10. Notifications

App permission ek baar maangega — Allow kar do. Ye milenge:
- Naye lesson upload
- Live class start
- Quiz / DPP reminder
- Doubt ka teacher reply

---

## 11. Help

- Koi dikkat ho to **Profile → Help / Support** se message bhejo.
- Refund / billing query bhi wahi se.

Padhai mubarak ho! 🙏
