## Deliverables

### (a) VariableSizeList + resetAfterIndex

```tsx
const listRef = useRef<VariableSizeList>(null);
const sizeMap = useRef<Record<number, number>>({});
const getSize = (i: number) => sizeMap.current[i] ?? ESTIMATED;
const setSize = (i: number, size: number) => {
  if (sizeMap.current[i] === size) return;
  sizeMap.current[i] = size;
  listRef.current?.resetAfterIndex(i);
};
```

Row uses `ResizeObserver` on its outer div â†’ calls `setSize(index, entry.contentRect.height)`.
Data mutation (delete, prepend, expand): `listRef.current?.resetAfterIndex(0)` after state commit.
