## 4. What's done right (wins worth keeping)

- `useAutoHideControls` is **exactly** what the master skill prescribes: `kick` / `forceHide` / `toggle` with `userHiddenRef` surviving lock flips. No drift.
- Tap-toggle deferred to `touchEnd` with `suppressTapToggleRef` set by every competing gesture (long-press, double-tap, swipe-lock, child-control hit) — textbook.
- Rotation-aware axis remap is present (see comments around `gVert`/`gHoriz` in the touch handlers).
- `installImmersiveAutoToggle()` MutationObserver owns nav-bar sync outside fullscreen — no fight with React.
- Black status-bar filler is the smallest possible diff: 6 lines, no library, no plugin.
- Floating chip uses `active:scale-90 transition-transform duration-150 ease-out` per soft-touch skill (no arbitrary `ms` values).
